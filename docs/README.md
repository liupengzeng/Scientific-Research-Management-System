# 项目文档目录

本目录用于存放职业院校科研管理系统的所有项目文档。

## 文档结构

### 📋 项目规划文档
- **plan/TASKS.md** - 项目实施计划（详细的任务清单和进度追踪，位于项目根目录）
- **项目进度总结.md** - 项目当前进度总结和下一步计划

### 🏗️ 系统架构文档
- **系统架构分析报告.md** - 完整的系统架构分析，包括前后端技术栈、架构设计等（位于项目根目录）

### 🔧 后端文档（建议移动到本目录）
- **research-backend/数据库初始化说明.md** - MySQL数据库初始化步骤和说明
- **research-backend/项目架构.md** - 后端项目架构说明
- **research-backend/项目说明.md** - 后端项目整体说明

### 🎨 前端文档（建议移动到本目录）
- **research-frontend/前端框架验证报告.md** - 前端框架验证结果报告
- **research-frontend/前端框架验证清单.md** - 前端框架验证步骤清单
- **research-frontend/前端配置验证说明.md** - 前端环境配置验证说明
- **research-frontend/前端项目结构图.md** - 前端项目结构说明
- **research-frontend/系统关键功能截图.md** - 系统关键功能界面截图说明

## 文档整理说明

由于PowerShell在处理中文文件名时可能存在编码问题，建议您：

1. **手动整理文档**：
   - 将 `research-backend/*.md` 移动到 `docs/` 目录
   - 将 `research-frontend/*.md`（除了README.md）移动到 `docs/` 目录
   - 将根目录的 `系统架构分析报告.md` 移动到 `docs/` 目录（可选）

2. **或者保持现状**：
   - 文档保留在各自目录中，通过文件名搜索查找

## 文档使用说明

1. **项目实施计划**: 查看 `plan/TASKS.md` 了解项目整体规划和当前进度
2. **项目进度**: 查看 `docs/项目进度总结.md` 了解当前进度和下一步计划
3. **系统架构**: 查看 `系统架构分析报告.md` 了解技术选型和架构设计
4. **开发指南**: 查看前后端的配置和验证文档，了解开发环境设置
